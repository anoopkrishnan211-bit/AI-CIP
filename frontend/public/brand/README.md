# ANIRA brand assets

- `anira-logo-horizontal.svg`: primary light-background lockup
- `anira-logo-white.svg`: single-color lockup for dark backgrounds
- `anira-logo-monochrome.svg`: print and grayscale version
- `anira-mark.svg`: standalone symbol for compact placements

The application favicon is maintained separately at `src/app/icon.svg` so Next.js can publish it automatically.

Core colors: navy `#061637`, blue `#2563EB`, violet `#7C3AED`, mint `#14D8A5`, and cyan `#22C7D9`.
